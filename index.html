Case Studies

Secure Authentication System

Context: Local Grown Salads is a sustainability-focused startup producing fresh salads and vegetables in-house. They needed an internal web platform to help employees securely access role-based dashboards related to farming, logistics, and team operations.

Problem: The company had no system in place to manage employee onboarding, access levels, or user authentication. They required a secure, scalable authentication and authorization solution that could grow alongside their cross-functional teams and operations.

My Role: I led the entire development of the authentication system from scratch. Collaborating closely with cross-functional teams—including ops managers, engineers, and team leads—I mapped out roles and access flows. I designed a Lucidchart diagram to visualize the user journey and developed backend services using Java and Python.

Solution: I built a secure login system using Spring Boot, Firebase Authentication, and JWT tokens. Role-based access was implemented to differentiate permissions between Managers and Employees. SQL was used for managing user metadata, while Firebase handled token validation. I also added API key functionality to support service integrations. To ensure reliability, I wrote test cases using JUnit and managed the build lifecycle and dependencies through Maven.

Impact: This solution reduced employee onboarding time by 40% and significantly improved login security. It gave the company a scalable foundation for access control and ensured that each user saw only what they were meant to—boosting productivity and safeguarding data integrity.

JioXplor: Real-Time Location Intelligence

Context: Reliance Jio is India’s largest telecom service provider, offering digital connectivity to hundreds of millions. I contributed to JioXplor, a geospatial intelligence platform designed to track and analyze location-based user activity across the country in real-time.

Problem: The team faced an enormous challenge: collecting and mapping high-volume geolocation data (latitude/longitude) from users across India and converting it into meaningful location insights. The process had to be fast, accurate, and scalable to power live business decisions and dashboards. The volume of data made traditional scraping and mapping methods inefficient, especially under real-time constraints.

My Role: I was deeply involved in building the geospatial engine behind JioXplor. I wrote Python and Shell scripts to automate data ingestion, and developed Spark (Scala) jobs to process data in distributed batches. I also worked with GIS tools and external APIs including Google Maps, HERE Maps, and OpenStreetMap, where I handled reverse geocoding, location tagging, and rate-limiting challenges.

Solution: To solve API rate limits and latency bottlenecks, I implemented multi-threaded request handling and orchestrated the process across multiple cloud VMs. I designed logic to throttle and retry failed requests efficiently. Processed data was mapped to Places of Interest (POIs), then stored in Akamai NetStorage and HDFS for redundancy. I queried location-tagged data using SQL to extract POI density and location-based trends. These insights were used by the business team for strategic decisions, including IPL 2022 customer engagement.

Impact: My solution enabled accurate geospatial mapping across India at scale. The business team gained access to millions of POI insights and real-time location patterns. This significantly boosted the quality of user targeting, infrastructure planning, and event-based decision-making. The system became a key data source for location-driven strategy and was praised for its speed, accuracy, and extensibility.

📚 Publications

“ARDUINO BASED SYSTEM TO MONITOR THE PLANTS' HEALTH AND GROWTH USING WIRELESS SENSORS”

Authors: Srujit Varasala, Mrs. Angline, Naveen Reddy, Shaik Khadeer Basha

Published in: Journal of Emerging Technologies and Innovative Research (JETIR), Volume 5, Issue 11, November 2018

Case Study:

In this research, we designed and implemented an IoT-based system using Arduino and wireless sensors to monitor the health and growth of plants in real-time. The system aims to address agricultural productivity challenges by enabling proactive plant care through data-driven insights.

The core functionality involved sensing nodes embedded in the soil, which captured key environmental metrics such as soil moisture, temperature, humidity, and even vibration patterns to detect larval activity. These nodes were powered by rechargeable batteries and communicated wirelessly with a base station using ZigBee protocol.

The base station aggregated the sensor data and relayed it to a mobile device via an Arduino processing unit. A dedicated mobile application then presented this data in a user-friendly format, allowing remote monitoring and timely alerts. This system could also automate irrigation based on moisture readings, enabling efficient resource usage and minimal manual intervention.

The solution is highly scalable and power-efficient, capable of supporting multiple sensors across larger agricultural fields. Its impact lies in reducing crop loss, improving food security, and making precision agriculture more accessible to small and medium-scale farmers.
